# FlaskLab
REST сервис на основе flask

Проект представляет собой сервис для загрузки объявлений и последующих действий над ними.
Каждое объявление хранится в базе данных instance/database.db и имеет следующие атрибуты:
+ Название объявления
+ Подробное описание
+ Дата написания
+ Владелец

В директории templates хранятся html документы, для фронта использовался bootstrap.

Приложение app.py реализовано с использованием flask. 
Приложение может работать с json и c html.
